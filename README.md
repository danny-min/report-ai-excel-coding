# 📊 报表生成 AI Demo

基于 Qwen/QwQ-32B 的智能报表生成工具核心链路验证

## 🏗️ 架构

```
用户需求描述
     ↓
┌────────────────┐
│  表格元信息     │  ← 自动解析表头、类型、示例数据
│  + 字段选择    │
│  + 需求描述    │
└────────────────┘
     ↓
┌────────────────┐
│  硅基流动 API   │  ← Qwen/QwQ-32B
│  生成 pandas   │
│  代码          │
└────────────────┘
     ↓
┌────────────────┐
│  安全执行器     │  ← 沙箱环境执行代码
└────────────────┘
     ↓
   结果预览
```

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

## 🚀 运行

### 自动演示模式
```bash
python demo.py
```

### 交互模式
```bash
python demo.py -i
```

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `config.py` | API 配置（密钥、模型名称等） |
| `api_client.py` | 硅基流动 API 调用封装 |
| `table_parser.py` | 表格解析模块 |
| `code_executor.py` | 安全代码执行器 |
| `demo.py` | 演示脚本 |

## 💡 使用示例

交互模式下可以输入类似这样的需求：

- "按区域统计总销售额"
- "统计每个产品的销售数量，按数量降序排列"
- "计算每个区域的平均销售额"
- "找出销售额大于1500的记录"

## ⚠️ 注意事项

1. API Key 已配置在 `config.py` 中
2. 代码执行有安全限制，不允许文件操作和系统调用
3. 生成的代码需要将结果存入 `result` 变量
