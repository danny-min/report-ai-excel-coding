# AI 报表生成器

基于 Electron + FastAPI + Pandas 的智能报表分析工具，通过自然语言描述即可完成数据分析和报表生成。

---

## 📋 功能清单

### ✅ 已实现功能

#### 1. 数据加载与管理
| 功能 | 说明 |
|------|------|
| Excel 文件上传 | 支持 `.xlsx`、`.xls` 格式 |
| CSV 文件上传 | 支持 UTF-8 编码 |
| 多表管理 | 同时加载多个数据表 |
| 表名智能简化 | 自动去除 UUID 后缀，保留核心名称 |
| 元数据提取 | 自动识别列名、数据类型、行数、Sheet 列表 |

#### 2. 自然语言分析
| 功能 | 说明 |
|------|------|
| 自然语言转 Pandas | 输入中文描述，自动生成可执行代码 |
| LLM 模型 | 使用硅基流动 Qwen2.5-Coder-32B-Instruct |
| 代码安全执行 | 沙箱环境，禁止危险操作 |
| 结果智能展示 | 根据数据类型自动选择展示方式 |

#### 3. 数据展示
| 类型 | 展示方式 |
|------|----------|
| DataFrame（表格） | 分页表格 + 自动条形图 |
| Series（序列） | 键值对表格 + 自动条形图 |
| 单值（数字/字符串） | 醒目大字展示 |

#### 4. 图表功能
| 功能 | 说明 |
|------|------|
| 前端条形图 | 纯 CSS 实现，自动渲染统计结果 |
| 导出柱状图 | Excel 原生图表，可编辑 |
| 导出饼图 | Excel 原生图表，可编辑 |
| 导出纯数据 | 无图表的 Excel |

#### 5. 多 Sheet 操作
| 功能 | 说明 |
|------|------|
| Sheet 预览 | 上传 Excel 时可查看所有 Sheet |
| 指定 Sheet 加载 | 选择要加载的具体 Sheet |
| 合并到 Sheet | 将一张表合并到另一张表的新 Sheet |
| 多表导出 | 将多张表导出到一个 Excel 的多个 Sheet |
| 自定义 Sheet 名 | 合并时可指定新 Sheet 的名称 |

#### 6. 用户体验
| 功能 | 说明 |
|------|------|
| 聊天式界面 | 对话形式交互 |
| 代码复制 | 一键复制生成的 Python 代码 |
| 反馈机制 | 👍/👎 评价分析结果 |
| 历史记录 | 保存查询历史，快速复用 |
| 状态栏 | 实时显示后端/LLM 连接状态 |

#### 7. 错误处理与防护
| 功能 | 说明 |
|------|------|
| EPIPE 防护 | 防止管道断开导致进程崩溃 |
| 全局异常捕获 | `uncaughtException` + `unhandledRejection` |
| 安全 Console | 重写 console.log/warn/error，防止写入错误 |
| JSON 安全序列化 | 处理 NaN/Inf 等特殊值 |
| Import 自动移除 | LLM 生成的 import 语句自动过滤 |

## 🚀 快速启动

### 1. 安装依赖

```bash
# Electron 依赖
cd electron-app
npm install

# Python 依赖
cd python-backend
pip install -r requirements.txt
```

### 2. 启动应用

```bash
# 开发模式（同时启动前后端）
npm run dev
```

### 3. 使用流程

1. **上传数据表** - 点击"上传文件"或"加载示例"
2. **输入需求** - 用自然语言描述分析需求
3. **查看结果** - 查看表格、图表、生成的代码
4. **导出报表** - 点击导出按钮下载 Excel

## 🔧 配置说明

### LLM API 配置

在 `python-backend/app.py` 中修改：

```python
SILICONFLOW_API_KEY = "你的API密钥"
MODEL_NAME = "Qwen/Qwen2.5-Coder-32B-Instruct"
```

### 端口配置

- **后端端口**: 5000（可通过环境变量 `PORT` 修改）
- **Electron**: 自动检测并连接后端
